What is this?
🤖
Purpose

AI Football Predictor is an experiment that pits multiple AI models against each other to see which one is best at predicting football match results. Every day, upcoming fixtures are fetched from football-data.org, enriched with the latest news, and sent as prompts to each configured model. The predictions are stored and scored once results are in, letting you compare model accuracy over time across leagues and matchdays.

How it works
⚙️
Prediction Pipeline
1
Fetch fixtures — upcoming matches are retrieved daily from the football-data.org API for all configured competitions.
2
Gather news — for each match, recent news articles and web results are fetched via the Brave Search API to give the models up-to-date context.
3
Prompt AI models — a structured prompt including competition details, team names, and news snippets is sent to every configured model simultaneously.
4
Store predictions — each model's predicted score and confidence are persisted to the database.
5
Score results — once a match finishes, the actual score is fetched and each prediction is scored automatically.
Scoring
🏆
How predictions are rated
3
Exact score
The predicted scoreline matches the actual result exactly (e.g. 2:1 → 2:1).
1
Correct tendency
The winner or draw is correct, but the scoreline is wrong (e.g. 2:0 → 1:0).
0
Wrong
Neither the scoreline nor the match outcome is correct.
